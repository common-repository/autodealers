== Description ==4.7
Door middel van deze plugin kunt u eenvoudig uw aanbod integreren binnen uw Wordpress website. U heeft hiervoor een account nodig van Autodealers. U kunt uzelf aanmelden via onze website www.autodealers.nl.

== Installation ==
1. Navigeer naar \'Plugins\' -> \'Add New\' en klick \'Upload Plugin\' selecteer daarna de .zip of pak de .zip uit en plaats de bestanden in de `/wp-content/plugins/` map
2. Activeer de plugin via het \'Plugins\' menu in WordPress
3. Navigeer naar \'Settings\' -> \'Autodealers\' of druk op Instellingen en vul het Dealer ID in dat u van Autodealers heeft gekregen
4. Plaats de shortcode [ad-voorraadlijst] op de gewenste plek
5. Bij gebruik [ad-carousel] of [ad-search-block] tevens Basis URL meegeven naar de URL waar de [ad-voorraadlijst] is geplaatst, vul deze URL aan met ?AdFrameSource=https://svl.autodealers.nl/
6. Parameters veld optioneel invullen bij instructie door Autodealers
